# 작업 보고서: 슬롯 업그레이드 기능

**날짜**: 2026-03-17
**작업자**: Claude

## 요청 내용

편성창의 잠긴 슬롯을 클릭하면 NoticePopup을 통해 확인 후 골드 100을 소모하여 슬롯을 확장하는 기능을 구현한다.

## 수행한 작업

- DeckPawnSlot에 `OnLockedClick` 이벤트를 추가하고, 잠긴 슬롯 좌클릭 시 발동하도록 `OnPointerClick` 분기 추가
- DeckManager에 `ExpandSlot()` 메서드를 추가하여 `_openSlotCount`를 1 증가 (최대 5)
- DeckSetting_Pawn에 `OnLockedSlotClick` 핸들러를 추가하여 NoticePopup 호출 → 골드 소모 → 슬롯 확장 → UI 갱신 처리
- 로드맵(99_로드맵.md)에서 해당 항목을 완료 처리
- 덱편성 기획서(10_덱편성_시스템.md)에 ExpandSlot API, OnLockedClick 이벤트, 슬롯 업그레이드 흐름 반영

## 주요 결정 사항

- **업그레이드 비용 100골드 고정**: 요구사항에 따라 임시 고정값으로 구현. `OnLockedSlotClick` 내 `const int upgradeCost = 100`으로 선언하여 추후 데이터화 시 변경 용이하도록 함
- **골드 부족 시 팝업은 닫히되 슬롯 확장 안 됨**: `SpendGold` 실패 시 return으로 처리. 별도 "골드 부족" 안내 팝업은 미구현

## 참고 사항

- 골드 부족 시 사용자에게 별도 피드백(안내 팝업 등)을 추가하면 UX 개선 가능
- 업그레이드 비용을 슬롯 단계별로 차등 적용하려면 `upgradeCost`를 데이터화 필요

# 작업 보고서: PawnStat 패널 구현

**날짜**: 2026-03-12
**작업자**: Claude

## 요청 내용

PawnManagePage에 추가된 PawnStat 그룹(GameObject)의 내용을 구현해달라는 요청. Pawn을 선택한 경우에만 `_goPawnStat`을 활성화하고, 선택된 Pawn의 스탯을 텍스트 필드에 표시해야 함.

## 수행한 작업

- `OnOpened()`에 `_goPawnStat.SetActive(false)` 추가 — 페이지 열릴 때 스탯 패널 초기 비활성화
- `OnPawnItemSelected()`에 `_goPawnStat.SetActive(true)` 및 `RefreshPawnStat()` 호출 추가 — Pawn 선택 시 패널 표시 및 스탯 반영
- `RefreshPawnStat()` 신규 메서드 추가 — 선택된 Pawn의 이름, 클래스, HP/ATK/DEF/SHIELD/MOVEMENT를 텍스트 필드에 적용
- `EquipToSlot()` 끝에 `RefreshPawnStat()` 호출 추가 — 장비 장착 후 스탯 즉시 갱신
- `OnSlotRightClick()` 끝에 `RefreshPawnStat()` 호출 추가 — 장비 해제 후 스탯 즉시 갱신

## 수정된 파일

|-----------|-----------|----------------|

## 주요 결정 사항

- **스탯 텍스트 형식**: `"레이블: 값"` 형식 사용 (예: `"HP: 100"`, `"ATK: 25"`) — 사용자 선택
- **스탯 값 출처**: `DPawn`의 런타임 프로퍼티(`HP`, `Attack`, `Armor`, `Shield`, `Movement`) 직접 사용 — 장비 보너스가 반영된 최종 값
- **이름/클래스 표시**: `PawnIcon.SetData()` 패턴 동일하게 재사용 (`pawn.CodeName`, `pawn.Data.ClassType.ToString()`)

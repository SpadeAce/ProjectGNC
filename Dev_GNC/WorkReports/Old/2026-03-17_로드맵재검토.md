## 요청 내용

- 로드맵의 각 항목을 실제 구현 상태와 대조하여 완료 여부를 검증함
- 완료된 항목 제거: 전투 결과 처리(5개), 장비 시스템(6개), 덱 Pawn 탭/슬롯 업그레이드, ShopPage/RecruitPage 기본 구현, 자원 UI 3개, 난이도 관리 2개, PawnManager 기술 부채
- 신규 항목 추가 (MEDIUM — 콘텐츠 구조): Pawn 클래스 차별화, 멀티 스테이지 구성, 랜덤 이벤트 시스템
- 신규 항목 추가 (LOW): OptionPopup 실제 구현 섹션
- 기술 부채 추가: Ammo 필드 네이밍 혼란 항목

- **완료 항목은 통째로 제거**: 별도 "완료" 섹션으로 이동하지 않고 깔끔하게 삭제함. 완료 이력은 git history와 기존 작업 보고서에서 추적 가능하므로 로드맵은 미구현 항목만 유지하는 방향으로 결정.
- **데이터 확장 항목 제외**: 카드 밸런스/종류 확장, 몬스터 종류 확장은 로드맵에 넣기엔 부적절하다는 사용자 판단에 따라 제외함.
- **신규 섹션 "콘텐츠 구조" 신설**: Pawn 클래스 차별화, 멀티 스테이지, 랜덤 이벤트는 기존 카테고리에 맞지 않아 MEDIUM 우선순위의 새 섹션으로 분리함.

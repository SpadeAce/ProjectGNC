# TileMap 에디터 툴 구현

- **일시**: 2026-03-28
- **분류**: 기능 구현 (에디터 툴)
- **범위**: Dev_GNCEditor 모듈 신규 생성 + TileMap 에디터 Slate 위젯

---

## 배경

Unity 프로젝트에는 `TileMapEditorWindow`라는 커스텀 에디터가 있어 비주얼 그리드로 타일맵을 편집할 수 있었으나, UE5 포팅 후 해당 툴이 없어 `UTileMapPreset` 데이터 에셋을 디테일 패널에서 수동 편집해야 하는 상황이었다.

## 구현 내용

### 1. Dev_GNCEditor 모듈 신규 생성

에디터 전용 C++ 모듈을 추가하여 에디터 빌드에서만 컴파일되도록 구성.

| 파일 | 역할 |
|------|------|
| `Source/Dev_GNCEditor/Dev_GNCEditor.Build.cs` | 모듈 빌드 규칙 (의존성: Dev_GNC, UnrealEd, ToolMenus 등) |
| `Source/Dev_GNCEditor/Dev_GNCEditor.h/.cpp` | 모듈 인터페이스, 도구 메뉴 등록, NomadTab 스포너 |

수정 파일:
- `Dev_GNC.uproject`: Dev_GNCEditor 모듈 항목 추가 (Type: Editor)
- `Dev_GNCEditor.Target.cs`: ExtraModuleNames에 Dev_GNCEditor 추가

### 2. TileMap 에디터 위젯 (Slate 기반)

`도구 > CardGame > TileMap Editor` 메뉴에서 접근 가능한 도킹 탭으로 구현.

| 파일 | 역할 |
|------|------|
| `TileMapEditorTypes.h` | 편집 모드 enum, 히트 테스트 구조체, 타일 색상 상수 |
| `STileMapEditorWindow.h/.cpp` | 메인 레이아웃 (설정/모드/도구/저장 패널) + 편집 로직 |
| `STileMapCanvas.h/.cpp` | 커스텀 OnPaint 그리드 렌더링 + 마우스 입력 처리 |

### 3. 주요 기능 (Unity 버전 대응)

| 기능 | 설명 |
|------|------|
| 그리드 생성 | 폭/높이 지정 후 Ground 타일로 초기화 |
| 타일 타입 페인팅 | Empty/Ground/Dirt/Water/Blocked 클릭/드래그 페인트 |
| 스폰 포인트 배치 | Player/Enemy + SpawnId 지정, 클릭 토글/드래그 페인트 |
| 엔티티 배치 | EntityCatalog 연동, 타일 셀 + 엣지 슬롯에 배치 |
| 전체 채우기/지우기 | 현재 모드의 선택값으로 일괄 적용/초기화 |
| 내보내기 | 콘텐츠 브라우저 저장 다이얼로그로 UTileMapPreset 데이터 에셋 생성 |
| 불러오기 | 에셋 피커로 기존 프리셋 로드 |
| 덮어쓰기 저장 | 연결된 프리셋에 현재 상태 즉시 반영 |

### 4. 크로스 모듈 링크

`UEntityCatalog` 클래스에 `DEV_GNC_API` 매크로를 추가하여 Dev_GNCEditor 모듈에서 심볼 접근 가능하도록 처리.

# Phase 5-C: 로비/타이틀 UI 구현

> **날짜**: 2026-03-25
> **작업 유형**: 기능 구현 (Unity→UE5 포팅)
> **Phase**: Phase 5-C — 로비/타이틀 UI

---

## 작업 요약

Unity의 TitleScript/TitlePage, LobbyScript/LobbyPage, LobbyManager를 UE5 C++로 포팅했다.
상점/모집 데이터 로직(ULobbySubsystem), 타이틀 페이지(UTitlePageWidget), 로비 페이지(ULobbyPageWidget)를 신규 생성하고, HUD 초기 페이지 자동 열기 기능과 테스트 데이터 생성 메서드를 기존 코드에 추가했다.

---

## 신규 파일 (6개)

### 1. ULobbySubsystem (`Subsystem/LobbySubsystem.h/cpp`)
- UGameInstanceSubsystem. Unity LobbyManager 1:1 매핑
- **상점**: GenerateShopList (ShopLevel→FShopDataRow.CardId→UDCard 생성, 최대 10개), LevelUpShop, RemoveShopCard
- **모집**: GenerateRecruitList (전체 PawnData에서 랜덤 3개 UDPawn 생성), RemoveRecruitPawn
- **유틸**: EnsureLobbyData (로비 진입 시 목록 자동 생성), ResetForNextStage, ResetAll
- **델리게이트**: FOnShopListChanged, FOnRecruitListChanged

### 2. UTitlePageWidget (`UI/TitlePageWidget.h/cpp`)
- UPageWidget 상속, UCLASS(Abstract)
- BindWidget 버튼 4개: NewGame, LoadGame, Option, Exit
- NewGame: 전 서브시스템(Player/Pawn/Deck/Item/Lobby) ResetAll → OpenLevel("Lobby")
- LoadGame: 현재 NewGame과 동일 (세이브/로드 미구현)
- Option: Phase 5-E 스텁, Exit: QuitGame

### 3. ULobbyPageWidget (`UI/LobbyPageWidget.h/cpp`)
- UPageWidget 상속, UCLASS(Abstract)
- BindWidget 버튼 6개: StartBattle, DeckSetting, PawnManage, Shop, Recruit, Exit
- BindWidgetOptional: TextGold, TextPawnCount
- NativeOnOpened: EnsureTestData → LobbySubsystem.EnsureLobbyData → RefreshInfoDisplay
- StartBattle: DeckSubsystem.GetActivePawns() 체크 → OpenLevel("Stage")
- DeckSetting/PawnManage/Shop/Recruit/Exit: Phase 5-D/5-E 스텁 (로그 출력)

---

## 수정 파일 (4개)

### 4. ACardGameHUD (`UI/CardGameHUD.h/cpp`)
- `InitialPageClass` (TSubclassOf\<UPageWidget\>) 프로퍼티 추가
- BeginPlay 끝에 InitialPageClass가 설정되어 있으면 자동으로 OpenPage 호출
- 레벨별 HUD Blueprint에서 초기 페이지를 지정하는 방식

### 5. UDeckSubsystem (`Subsystem/DeckSubsystem.h/cpp`)
- `InitTestData()` 메서드 추가
- Unity DeckManager.InitTestData() 동일 구현: 카드 6장(10001×3, 10004×3) + 폰 2개(PawnId=1) + 장비 11개 + 슬롯 0,1에 편성

## Unity → UE5 매핑

| Unity | UE5 |
|-------|-----|
| TitleScript.OnEnterScene → TitlePage 열기 | ACardGameHUD.InitialPageClass 자동 열기 |
| LobbyScript.OnEnterScene → 테스트 데이터 + LobbyPage | ULobbyPageWidget.EnsureTestData + NativeOnOpened |
| LobbyManager (MonoSingleton) | ULobbySubsystem (UGameInstanceSubsystem) |
| SceneController.ChangeScene() | UGameplayStatics::OpenLevel() |
| SceneController.ReturnToTitle + ResetAllManagers | UTitlePageWidget.ResetAllSubsystems |
| DeckManager.InitTestData() | UDeckSubsystem::InitTestData() |

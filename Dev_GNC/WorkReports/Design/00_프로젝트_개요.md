# 프로젝트 개요

## 게임 장르 및 컨셉

턴제 전술 카드 게임. 타일맵 위에서 폰(Pawn)을 이동시키고 카드를 사용해 몬스터를 처치하는 방식의 전략 RPG.

---

## 기술 스택

| 항목 | 내용 |
|------|------|
| 엔진 | Unreal Engine 5 |
| 언어 | C++ |
| 데이터 직렬화 | DataTable (FTableRowBase) |
| 데이터 저장 | DataTable `.csv` → `.uasset` (Content 폴더) |
| 에셋 관리 | TSoftClassPtr / TSoftObjectPtr, StaticLoadObject |

## Subsystem 패턴

### UGameInstanceSubsystem (게임 전체 수명)
- 게임 인스턴스와 동일한 수명 — 레벨 전환에도 유지됨
- `GetGameInstance()->GetSubsystem<T>()` 로 접근
- **사용**: UDataSubsystem, UTextSubsystem, UPlayerDataSubsystem, UPawnSubsystem, UItemSubsystem, UDeckSubsystem, ULobbySubsystem (7개)

### UWorldSubsystem (레벨 수명)
- 레벨과 동일한 수명 — 레벨 전환 시 자동 소멸/재생성
- `GetWorld()->GetSubsystem<T>()` 로 접근
- **사용**: UGridSubsystem, UTurnSubsystem, UStageSubsystem (3개)

## 레벨(맵) 목록

| 레벨 이름 | GameMode | 용도 |
|-----------|----------|------|
| Title | ACardGameMode (BP_CardTitleGameMode) | 타이틀 화면 |
| Lobby | ACardGameMode (BP_CardLobbyGameMode) | 로비 (덱 세팅, 상점 등) |
| Stage | ACardStageGameMode (BP_CardStageGameMode) | 전투 진행 |

> Unity의 BootScene은 불필요 — UE5 DataTable은 엔진이 자동 로드하며, UDataSubsystem::Initialize()에서 초기화.

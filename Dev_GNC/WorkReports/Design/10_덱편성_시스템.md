# 덱 편성 시스템 (DeckSetting)

## 구현 상태: Pawn 탭 완성, Card 탭 기본 구현 완료

---

## 핵심 파일

| 파일 | 역할 |
|------|------|
| `UI/DeckSettingPageWidget.h/cpp` | 덱 편성 페이지 — Pawn/Card 탭 전환 |
| `UI/PawnListItemWidget.h/cpp` | 보유 Pawn 아이템 위젯 |
| `Subsystem/DeckSubsystem.h/cpp` | 편성 슬롯(DeckPawnSlots) + 카드 덱(DeckCardList) 관리 |

---

## UDeckSubsystem Pawn 관련 API

| 메서드/프로퍼티 | 설명 |
|----------------|------|
| `GetOwnPawns()` | 보유 전체 Pawn 목록 |
| `GetDeckPawns()` | 편성된 Pawn 목록 (null 슬롯 제외, Stage용) |
| `OpenSlotCount` (int32, 초기값 2) | 열린 슬롯 수 |
| `ExpandSlot()` | 슬롯 1개 확장 (최대 5), 성공 시 true |
| `AddPawn(UDPawn*)` | 영입한 Pawn을 보유 목록에 추가 |
| `AssignPawnToDeck(UDPawn*, int32)` | 슬롯에 Pawn 편성 |
| `UnassignPawnFromDeck(int32)` | 슬롯 편성 해제 |
| `FindEmptySlotIndex()` | 첫 번째 빈 슬롯 인덱스 반환 (-1: 없음) |
| `GetDeckPawn(int32)` | 슬롯 인덱스로 편성된 Pawn 조회 |

```cpp
// 내부 구조
TArray<UDPawn*> OwnPawnList;       // 보유 전체
UDPawn* DeckPawnSlots[5];          // 편성 슬롯
int32 OpenSlotCount = 2;           // 열린 슬롯 수 (추후 확장 가능)
```

## Pawn 편성 흐름

```
UDeckSettingPageWidget::OnOpened()
    └── Pawn 탭 초기화
            ├── InitSlots()        — 슬롯 잠금 여부 설정, 기존 편성 표시
            └── RefreshOwnPawns()  — 보유 Pawn → PawnListItemWidget 생성

PawnListItem 드래그 → 슬롯에 드롭
    └── 슬롯 탐색
    └── UDeckSubsystem::AssignPawnToDeck() + 슬롯 표시 갱신

PawnListItem 우클릭
    └── UDeckSubsystem::FindEmptySlotIndex()
    └── UDeckSubsystem::AssignPawnToDeck() + 슬롯 표시 갱신

슬롯 우클릭
    └── UDeckSubsystem::UnassignPawnFromDeck() + 슬롯 표시 초기화
```

---

## 슬롯 잠금 정책

| 슬롯 | 초기 상태 | 조건 |
|------|----------|------|
| 0, 1 | 열림 | 기본 제공 |
| 2, 3, 4 | 잠김 | 추후 업그레이드로 개방 예정 |

- `UDeckSubsystem::OpenSlotCount` 값을 늘리면 추가 슬롯 개방
- 잠긴 슬롯은 드래그 드롭 불가 (`AssignPawnToDeck` 내부에서 검증)
- 잠긴 슬롯 좌클릭 → UNoticePopupWidget으로 확인 → 100골드 소모 → `UDeckSubsystem::ExpandSlot()` → UI 갱신

## UDeckSubsystem Card 관련 API

| 메서드/프로퍼티 | 설명 |
|----------------|------|
| `GetDeckCards()` | 보유 전체 카드 목록 (읽기 전용) |
| `AddCard(UDCard*)` | 카드를 덱에 추가 |

```cpp
// 내부 구조
TArray<UDCard*> DeckCardList;   // 보유 카드 전체
```

---

## Card 탭 흐름

```
Card 탭 버튼 클릭
    └── Card 탭 초기화
            └── RefreshCardList()  — DeckCards → 카드 아이템 위젯 생성 (5열 그리드)

카드 아이템 위젯.SetData(UDCard*)
    └── 카드 정보 표시
```

- 수직 스크롤, 한 줄에 5장 표시
- Content 높이를 행 수에 따라 동적 조정

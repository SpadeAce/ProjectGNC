## ACardGameHUD

- **패턴**: AHUD 상속, ACardGameMode 생성자에서 HUDClass로 지정
- **레벨별 설정**: GameMode BP의 Details 패널에서 `InitialPageClass`, `FloatingTextClass` 등 설정

### UMG 레이어 구조
```
ACardGameHUD
├── PageLayer     : UOverlay — PageWidget 전용 레이어 (하위)
├── PopupLayer    : UOverlay — PopupWidget 전용 레이어 (상위)
└── FloatingText 풀 관리
```

### 주요 메서드
```cpp
void OpenPage(TSubclassOf<UPageWidget> PageClass)
    → PageLayer에 CreateWidget → AddChildToOverlay
    → 이전 top 페이지가 bDontHide=false이면 SetVisibility(Collapsed)
    → PageStack에 Push → PreOpen() → OnOpened() 순서로 호출

void ClosePage(UPageWidget* Page)
    → PreClose() → PageStack에서 제거 → RemoveFromParent()
    → 이전 top 페이지 SetVisibility(Visible) 복원

void OpenPopup(TSubclassOf<UPopupWidget> PopupClass)
    → PopupLayer에 CreateWidget → AddChildToOverlay
    → PreOpen() → OnOpened()

void ClosePopup(UPopupWidget* Popup)
    → PreClose() → RemoveFromParent()

void CloseAllPages()
    → 열린 모든 Page PreClose() 후 RemoveFromParent()
    → PageStack Clear

void CloseAllPopups()
    → 열린 모든 Popup PreClose() 후 RemoveFromParent()
```

## Page 전환 동작

PageWidget은 스택 기반으로 전환된다. 새 Page가 열리면 이전 Page는 숨겨지고, 닫히면 이전 Page가 복원된다.

```
LobbyPage 열림
  └─ PageStack: [LobbyPage]

ShopPage 열림
  └─ LobbyPage.SetVisibility(Collapsed)
  └─ PageStack: [LobbyPage, ShopPage]

ShopPage 닫힘
  └─ ShopPage RemoveFromParent
  └─ LobbyPage.SetVisibility(Visible)
  └─ PageStack: [LobbyPage]
```

### bDontHide 옵션
```cpp
// UPageWidget
UPROPERTY(EditDefaultsOnly)
bool bDontHide = false;
```
- `false` (기본값): 새 Page 오픈 시 이 페이지가 숨겨짐
- `true`: 새 Page가 열려도 이 페이지는 숨겨지지 않고 유지됨

## View 라이프사이클

```
OpenPage/OpenPopup 호출
    → PreOpen()    : 버튼 리스너 등록, Delegate 구독, 데이터 세팅
    → OnOpened()   : 초기 표시 업데이트 (카드 목록 갱신 등)

ClosePage/ClosePopup 호출
    → PreClose()   : Delegate 구독 해제, 리스너 제거
    → RemoveFromParent() → GC 회수
```

## 각 뷰 상세

### UTitlePageWidget
- New Game → `OpenLevel("Lobby")`
- Load Game → `OpenLevel("Lobby")`
- Option → OptionPopup
- Exit → `QuitGame()`

### ULobbyPageWidget
- Start Battle → `OpenLevel("Stage")`
- Deck Setting → DeckSettingPageWidget 오픈
- Shop → ShopPageWidget 오픈
- Recruit → RecruitPageWidget 오픈
- Exit → 전체 Subsystem ResetAll() + `OpenLevel("Title")`

### UShopPageWidget
- ULobbySubsystem의 ShopLevel 기반으로 최대 10개 카드 진열 (5×2 그리드)
- 구매 시 ShopItem 숨김, UDeckSubsystem::AddCard()으로 UDCard 추가
- LevelUp 버튼으로 최고 레벨까지 단계 상승 가능
- Stage 진행 전까지 진열 목록 유지 (Refresh() 호출 시 갱신)
- Close → ClosePage() → LobbyPage 복원
- **관련 파일**: `09_상점_시스템.md`

### UDeckSettingPageWidget
- Pawn/Card 탭 전환 버튼 구현
- **Pawn 탭**: 보유 Pawn 목록 표시, 드래그 or 우클릭으로 슬롯 편성
- **Card 탭**: 미구현
- Close → ClosePage() → LobbyPage 복원
- **관련 파일**: `10_덱편성_시스템.md`

### URecruitPageWidget
- Close → ClosePage() → LobbyPage 복원

### UPawnManagePageWidget
- 좌측 스크롤 리스트: 보유 Pawn을 UPawnListItemWidget으로 표시 (가로 스크롤)
- 우측 스크롤 리스트: 보유 장비를 UEquipListItemWidget으로 4열 그리드 표시 (세로 스크롤)
- 장비 슬롯 영역: Weapon, Armor, Tool×2 총 4슬롯

#### 인터랙션
| 액션 | 결과 |
|------|------|
| PawnListItem 좌클릭 | 해당 Pawn 선택, 선택 표시 활성화, 슬롯 갱신 |
| EquipListItem 드래그 → 슬롯 릴리즈 | 슬롯 타입 일치 시 선택 Pawn에 장착 |
| EquipListItem 우클릭 | 해당 타입의 슬롯에 자동 장착 (빈 슬롯 우선) |
| EquipSlot 우클릭 | 장착 해제 |

- Close → ClosePage() → LobbyPage 복원

### UStagePageWidget
- **구독 Delegate**: UDeckSubsystem::OnHandChanged, UStageSubsystem::OnBattleEnd, UStageSubsystem::OnSelectedChanged, UTurnSubsystem::OnTurnChanged, UTurnSubsystem::OnSharedActingPowerChanged
- **글로벌 카드 UI** (CardGroup): UDeckSubsystem::HandCardList → UBattleCardWidget 동적 생성
- **Pawn 카드 UI** (PawnCardGroup): 선택된 Pawn의 PawnHandCards → UBattleCardWidget 동적 생성
  - Pawn 선택 시 해당 Pawn의 OnPawnHandChanged Delegate 구독 (선택 해제 시 구독 해제)
- **버튼**: Pause(UPausePopupWidget), EndTurn(UTurnSubsystem::EndPawnTurn)
- **전투 결과 수신**: UStageSubsystem::OnBattleEnd → UBattleResultPopupWidget 열기

### UBattleResultPopupWidget
- **공통**: 골드/경험치 텍스트 표시
- **승리 시**: Continue 버튼 표시, 카드 보상 3장 세팅
  - FStageRewardRow 가중치 랜덤 3장 (중복 불가)
  - 클릭으로 1장 선택/해제
  - Continue 클릭 → 골드/경험치 적용, 선택 카드 추가, `OpenLevel("Lobby")`
- **패배 시**: Exit 버튼 표시
  - Exit 클릭 → 전체 Subsystem ResetAll() + `OpenLevel("Title")`
- Title 텍스트: "승리" / "패배"

### UPausePopupWidget
- Continue → ClosePopup()
- Exit → 전체 Subsystem ResetAll() + `OpenLevel("Title")`

### UNoticePopupWidget
- 범용 확인/취소 팝업
- Confirm/Cancel 버튼 + 콜백

## HUD 시스템

### UFloatingTextWidget (플로팅 텍스트)
- **파일**: `UI/FloatingTextWidget.h/cpp`
- **Widget BP**: `WBP_FloatingText`
- **용도**: 전투 중 데미지/회복/버프 등 수치 변화를 액터 위에 떠오르는 텍스트로 표시
- **호출 경로**: UDPawn/UDMonster OnFloatingText Delegate → ACardGameActor → ACardGameHUD::ShowFloatingText()

#### EFloatingTextType

| 종류 | 색상 | 연출 | 트리거 |
|------|------|------|--------|
| Damage | 빨강 | 대각 포물선 + 바운스 | `TakeDamage()` (net > 0) |
| Heal | 초록 | 수직 상승 | `Heal()` |
| Shield | 하늘 | 수직 상승 | `AddShield()` |
| Buff | 주황 | 제자리 팝 | `ApplyBuff(Buff*)` |
| Debuff | 보라 | 제자리 팝 | `ApplyBuff(Debuff*)` |
| Block | 회색 | 제자리 팝 | `TakeDamage()` (net == 0) |

#### 겹침 방지
- ACardGameHUD가 액터별 활성 텍스트 수를 추적
- 동일 액터에 연속 텍스트 발생 시 Y축 오프셋 누적 (0.4 단위)
- 텍스트 완료 시 자동으로 카운트 감소

#### 오브젝트 풀
- 초기 10개 생성, 부족 시 자동 확장
- 비활성 인스턴스를 TArray/TQueue로 관리

## UBattleCardWidget (전투 카드 UI)
- **파일**: `UI/BattleCardWidget.h/cpp`
- **Widget BP**: `WBP_BattleCard`
- **기능**: 전투 전용 카드 UI — 표시 + 드래그&드롭 이벤트 발행
- **드래그**: UMG Native Drag 시스템 (UCardDragDropOp) 사용
- **표시 필드**: 비용(Cost), 이름(Name), 설명(Desc), 아이콘 이미지, Dimmed 오버레이(비활성화)

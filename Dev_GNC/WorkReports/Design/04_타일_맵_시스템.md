# 타일 맵 시스템

## 구현 상태: 완성

## 핵심 파일

| 파일 | 역할 |
|------|------|
| `Subsystem/GridSubsystem.h/cpp` | 타일맵 생성, 경로탐색, 좌표 변환 |
| `Grid/SquareTile.h/cpp` | 개별 타일 AActor — 상태 및 시각화 |
| `Grid/EdgeTile.h/cpp` | 타일 간 경계 AActor (벽 배치용) |

| `Data/TileMapPreset.h/cpp` | 맵 구성 데이터 (UDataAsset) |

## 좌표 시스템

- 그리드: `FIntPoint(X, Y)` → 월드 좌표 X, Y 축
- 타일 간격: `TileSize` (기본 1.05f)
- 4방향: North(0,1) / East(1,0) / South(0,-1) / West(-1,0)

---

## 타일 종류 (ETileType)

| 값 | 이름 | 이동 가능 | 설명 |
|----|------|----------|------|
| Empty | 빈 공간 | X | 타일 없음 |
| Ground | 지면 | O | 일반 지형 |
| Dirt | 흙 | O | 흙 지형 |
| Water | 물 | X | 이동 불가 |
| Blocked | 막힘 | X | 차단된 타일 |

---

## 타일 상태 (ETileState)

| 상태 | 설명 |
|------|------|
| Normal | 기본 |
| Selected | 클릭 선택됨 (노란 테두리 등) |
| Highlighted | 이동 가능 범위 표시 (파란 하이라이트) |
| Disabled | 비활성화 |

추가 시각화: `SetTarget(bool)` — 카드 사거리 표시 (빨간 하이라이트)

## AEdgeTile (타일 간 경계)

- 인접한 두 타일 사이의 경계에 배치하는 AActor
- 키: `FEdgeTileKey(FIntPoint PosA, FIntPoint PosB)`
- 용도: 벽(`ATileWall`) 배치 → 경로탐색에서 통과 불가로 처리
- 방향: North / East / South / West

---

## UTileMapPreset (UDataAsset)

- **위치**: `Content/CardGame/DataAssets/`
- **필드**:
  ```
  int32 Width, Height      : 맵 크기
  float TileSize           : 타일 간격 (기본 1.05f)
  int32 TurnLimit          : 턴 제한 (0 = 무제한)
  int32 MaxPawnCount       : 배치 가능한 최대 Pawn 수
  TArray<FTilePresetData> Tiles  : 각 타일 설정
  TArray<FEdgePresetData> Edges  : 경계(벽) 설정
  ```
- **FTilePresetData 필드**:
  ```
  FIntPoint Position       : 그리드 위치
  ETileType TileType       : 타일 종류
  ESpawnPointType SpawnPoint : 스폰 포인트 종류 (Player/Enemy)
  int32 SpawnId            : Pawn ID 또는 Monster ID
  int32 EntityId           : EntityCatalog ID (타일 오브젝트)
  ```
- **FEdgePresetData 필드**:
  ```
  FIntPoint PosA, PosB     : 두 타일 위치
  int32 EntityId           : 배치할 엔티티 (벽 등)
  ```

## 경로탐색 알고리즘

### GetReachableTiles() — 도달 가능 범위
- **알고리즘**: BFS (너비 우선 탐색)
- **이동 규칙**:
  - Water / Blocked 타일 진입 불가
  - AEdgeTile에 ATileWall이 있으면 통과 불가
  - 같은 진영 유닛 타일 통과 가능 (단, 정지 불가)
  - 다른 진영 유닛 타일 통과/진입 불가

### GetPath() — 경로 계산
- **알고리즘**: A* (맨해튼 거리 휴리스틱)
- **이동력 제한**: Movement 이내 경로만 반환

### SmoothPath() — 경로 스무딩
- **알고리즘**: Greedy 직선 시야 검사
- 웨이포인트를 최소화하여 자연스러운 이동 경로 생성
- `HasLineOfSight()` — Bresenham 직선 알고리즘 + 코너 커팅 방지

## 맵 생성 흐름

```
ACardStageGameMode::LoadStageFromPreset()
    └─▶ UGridSubsystem::GenerateTileMapFromPreset(Preset)
              ├─ 각 FTilePresetData → ASquareTile SpawnActor
              ├─ 각 FEdgePresetData → AEdgeTile + ATileWall 배치
              └─ SpawnPoint 타일 반환
    └─▶ PlayerSpawn 위치 → ACardGameActor(UDPawn) 스폰
    └─▶ EnemySpawn 위치 → ACardGameActor(UDMonster) 스폰
```

## UEntityCatalog (UDataAsset)

- **위치**: `Content/CardGame/DataAssets/EntityCatalog`
- **용도**: EntityId → 블루프린트 클래스 매핑
- **사용**: UTileMapPreset의 EntityId로 ATileWall 등 배치

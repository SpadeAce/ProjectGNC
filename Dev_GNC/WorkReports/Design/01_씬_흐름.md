## 레벨 전환 흐름

```
Title (레벨)
    │ [New Game / Load Game 버튼]
    └──▶ Lobby (레벨)
              │ 최초 진입 시 DeckSubsystem.InitTestData()
              │ [Start Battle 버튼]
              └──▶ Stage (레벨)
                        │ [BattleResultPopup - 승리]
                        │  PlayerDataSubsystem.IncrementLevel()
                        ├──▶ Lobby
                        │ [BattleResultPopup - 패배]
                        └──▶ Title (ReturnToTitle)
                                ↑ 전체 Subsystem ResetAll() 실행
```

> Unity의 BootScene은 불필요 — UE5에서는 DataTable을 엔진이 자동 로드하므로 별도 부팅 씬이 필요 없음. UDataSubsystem::Initialize()에서 DataTable 참조를 초기화.

## 레벨별 상세

### Title
- **GameMode**: `BP_CardTitleGameMode` (Parent: ACardGameMode)
- **HUD InitialPageClass**: `WBP_TitlePage`
- **UI**: UTitlePageWidget
- **버튼**:
  | 버튼 | 동작 |
  |------|------|
  | New Game | `UGameplayStatics::OpenLevel("Lobby")` |
  | Load Game | `UGameplayStatics::OpenLevel("Lobby")` (현재 Save/Load 미구현) |
  | Option | OptionPopup 열기 |
  | Exit | `UKismetSystemLibrary::QuitGame()` |

---

### Lobby
- **GameMode**: `BP_CardLobbyGameMode` (Parent: ACardGameMode)
- **HUD InitialPageClass**: `WBP_LobbyPage`
- **UI**: ULobbyPageWidget
- **BeginPlay 동작**:
  1. 보유 Pawn이 없으면 `UDeckSubsystem::InitTestData()` — 테스트 데이터 초기화
  2. ACardGameHUD가 InitialPageClass(WBP_LobbyPage)를 자동 생성
- **버튼**:
  | 버튼 | 동작 | 구현 상태 |
  |------|------|---------|
  | Start Battle | `OpenLevel("Stage")` (DeckPawns 없으면 무시) | 완성 |
  | Deck Setting | DeckSettingPageWidget 오픈 | 완성 (Pawn 탭, Card 탭 미구현) |
  | Pawn Manage | PawnManagePageWidget 오픈 | 완성 |
  | Shop | ShopPageWidget 오픈 | 완성 |
  | Recruit | RecruitPageWidget 오픈 | 완성 |
  | Exit | 전체 Subsystem ResetAll() + `OpenLevel("Title")` | 완성 |

---

### Stage
- **GameMode**: `BP_CardStageGameMode` (Parent: ACardStageGameMode)
- **HUD InitialPageClass**: `WBP_StagePage`
- **UI**: UStagePageWidget
- **ACardStageGameMode::BeginPlay() 동작**:
  1. ACardGameHUD가 InitialPageClass(WBP_StagePage) 자동 생성
  2. `LoadStageFromPreset()` — 난이도 레벨에 맞는 UTileMapPreset으로 맵 생성 및 Actor 스폰
     - `UPlayerDataSubsystem::DifficultyLevel` → `UDataSubsystem::GetStagePresetByLevel()` → `PresetPath` 랜덤 선택 → `StaticLoadObject<UTileMapPreset>()`
  3. `SetCameraPivot()` — UGridSubsystem::GetMapCenter() → CameraPawn::SetPivot()
  4. UDeckSubsystem::InitStage() — 카드풀 셔플 + 초기 핸드 5장 드로우
  5. UTurnSubsystem::StartGame() — Pawn 턴 시작

## 레벨 전환 메커니즘

- **UE5 레벨 전환**: `UGameplayStatics::OpenLevel(GetWorld(), "LevelName")`
- OpenLevel 호출 시 현재 World가 소멸되므로 HUD, 위젯, WorldSubsystem이 자동 정리됨
- **별도 UI 정리 불필요**: OpenLevel이 World를 파괴하면서 ACardGameHUD 포함 모든 위젯이 자동 소멸

### ReturnToTitle 처리
```
UGameplayStatics::OpenLevel("Title") 호출 전:
    → UPlayerDataSubsystem::ResetAll()   — 난이도=1, 골드=1000
    → UPawnSubsystem::ResetAll()         — Pawn 목록 초기화
    → UDeckSubsystem::ResetAll()         — 슬롯/카드 리스트 초기화
    → UItemSubsystem::ResetAll()         — 장비 목록 초기화
    → ULobbySubsystem::ResetAll()        — 상점/영입 상태 초기화
    → UGameplayStatics::OpenLevel("Title")
```

- GameInstanceSubsystem은 레벨 전환에도 유지되므로, 명시적으로 ResetAll() 호출 필요
- WorldSubsystem(Grid, Turn, Stage)은 레벨 전환 시 자동 소멸/재생성되므로 별도 리셋 불필요
